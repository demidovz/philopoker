# 🧠 Настройка нейросетевых ИИ-игроков

Игра теперь использует **OpenRouter** вместо простых алгоритмов для реалистичного поведения ИИ-философов.

## 🚀 Быстрая настройка

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Настроить API ключ
```bash
# Скопируйте образец конфигурации
cp .env.example .env

# Отредактируйте .env файл
nano .env
```

В файле `.env` укажите ваш OpenRouter API ключ:
```env
OPENROUTER_API_KEY=sk-or-v1-your-openrouter-key-here
AI_MODEL=nvidia/nemotron-3-super-120b-a12b:free
MAX_TOKENS=150
TEMPERATURE=0.7
USE_FALLBACK=true
```

### 3. Получить API ключ OpenRouter
1. Зайдите на https://openrouter.ai/
2. Создайте API ключ
3. Скопируйте его в `.env` файл или сохраните в `../openrouter.key`

## 🎭 Типы нейросетевых личностей

### 🤔 Скептик (Skeptic)
- Критически анализирует утверждения
- Ищет противоречия и слабые места
- Задает острые вопросы
- Высокая склонность к несогласию

### 📚 Педант (Pedant)
- Требует точности определений
- Акцентирует на академической корректности
- Ссылается на источники и авторитеты
- Методичный в аргументации

### 🛠️ Прагматик (Pragmatist)
- Оценивает практическую пользу
- Фокусируется на реальных последствиях
- Приводит конкретные примеры
- Избегает абстрактных рассуждений

### 🔄 Синтезатор (Synthesizer)
- Ищет компромиссы между позициями
- Объединяет разные точки зрения
- Сглаживает конфликты
- Предлагает альтернативные решения

## ⚡ Как работает ИИ

### Оценка утверждений
```python
# Нейросеть анализирует утверждение и возвращает:
{
  "position": "agree" | "disagree",
  "confidence": 0.1-1.0,
  "reasoning": "краткое обоснование"
}
```

### Генерация вопросов
- ИИ создает критические вопросы в стиле своей личности
- Учитывает контекст дискуссии
- Максимум 15 слов для краткости

### Аргументация
- ИИ формулирует убеждающие аргументы
- Адаптируется к позиции (за/против)
- Максимум 25 слов для четкости

### Финальное голосование
- ИИ анализирует всю дискуссию
- Оценивает убедительность аргументов
- Принимает беспристрастное решение

## 🛡️ Система отказоустойчивости

Если OpenRouter недоступен, игра автоматически переключается на **fallback режим**:

### Fallback логика
- ✅ Простые алгоритмические ответы
- ✅ Предзаданные фразы и вопросы
- ✅ Случайное голосование
- ✅ Игра продолжается без остановки

### Индикаторы работы
- `🤖 Нейросеть активна` - OpenRouter API работает
- `⚠️ Fallback режим` - используется резервная логика
- `🔥 Ошибка API` - временные проблемы с сетью

## 💰 Стоимость использования

### OpenRouter: NVIDIA Nemotron 3 Super (free)
- **~$0.002** за 1000 токенов (входящих)
- **~$0.002** за 1000 токенов (исходящих)

- **Стоимость 1 игры** ≈ $0.01-0.02

### Оптимизация расходов
- Используйте `MAX_TOKENS=150` для экономии
- Включите `USE_FALLBACK=true` как резерв
- Ограничьте `TEMPERATURE=0.7` для стабильности

## 🔧 Продвинутая настройка

### Изменение модели
```env
# Более мощная (дорогая) модель
AI_MODEL=gpt-4

# Более экономная модель
AI_MODEL=gpt-3.5-turbo-instruct
```

### Настройка поведения
```env
# Более креативные ответы
TEMPERATURE=0.9

# Более предсказуемые ответы
TEMPERATURE=0.3

# Длинные развернутые ответы
MAX_TOKENS=300
```

## 🐛 Решение проблем

### API ключ не работает
- Проверьте правильность ключа
- Убедитесь в наличии средств на счете
- Проверьте лимиты использования

### Медленные ответы
- Уменьшите `MAX_TOKENS`
- Проверьте интернет-соединение
- Попробуйте другую модель

### Ошибки rate limit
- Игра автоматически делает паузы
- При превышении лимитов включается fallback
- Проверьте лимиты и доступность вашего ключа OpenRouter
