# 🎯 Многоуровневый сократовский метод

Игра теперь реализует **истинный многоуровневый сократовский диалог** с рекурсивным углублением.

## 🔄 Как работает новая механика

### **📝 ФАЗА 1: Сбор критических вопросов**
Все игроки с позицией "disagree" задают свои вопросы:
```
❓ 1. Скептик: А как объяснить культурные различия в понимании красоты?
❓ 2. Игрок: Почему дети и взрослые видят красоту по-разному?
❓ 3. Педант: Где научные критерии объективной красоты?
```

### **🎯 ФАЗА 2: Выбор главного вопроса**
**Человек-игрок выбирает** самый сильный вопрос из списка:
```
Выберите номер лучшего вопроса (1-3): 2
📌 Выбран вопрос #2: Почему дети и взрослые видят красоту по-разному?
```

### **🛡️ ФАЗА 3: Ответ защитника**
Главный защитник (agree с макс. ставкой) отвечает:
```
🛡️ Защитник: Это не противоречит объективности - просто развивается восприятие
```

### **🤔 ФАЗА 4: Проверка углубления**
Игроки решают, хотят ли оспорить **ответ**:
```
Игрок, хотите оспорить ответ и углубиться? (да/нет): да
🔍 Игрок: Хочет углубиться
```

### **🔄 УРОВЕНЬ 2: Рекурсивное углубление**
Теперь обсуждаем **ответ** как новое утверждение:
```
🎯 СОКРАТОВСКИЙ ДИАЛОГ (Уровень 2):
📝 Углубляемся в: "Это не противоречит объективности - просто развивается восприятие"

❓ 1. Игрок: Как "развитие восприятия" совместимо с "объективностью"?
❓ 2. Скептик: Если восприятие меняется, то где объективная основа?
```

### **⚡ И так далее до 3 уровней максимум**

## 🎮 Стратегические элементы

### **Для критиков (disagree):**
- **Задавайте острые вопросы** выявляющие противоречия
- **Участвуйте в выборе** лучшего вопроса
- **Углубляйтесь в ответы** если видите слабые места

### **Для защитника (agree):**
- **Высокая ставка** = высокая ответственность за защиту
- **Отвечайте осторожно** - ваш ответ может стать новым объектом критики
- **Готовьтесь к многоуровневой защите**

### **Для всех игроков:**
- **Финальное голосование** учитывает ВСЕ уровни диалога
- **Качество ответов важнее** их количества
- **Интеллектуальная честность** ценится больше упрямства

## 🧠 Философская ценность

### **Развивает навыки:**
1. **Формулирования острых вопросов** (критическое мышление)
2. **Защиты позиций под давлением** (аргументация)
3. **Выявления слабых мест** в рассуждениях (логический анализ)
4. **Многоуровневого анализа** (глубина понимания)

### **Принципы Сократа:**
- **"Я знаю, что ничего не знаю"** - готовность пересматривать позиции
- **Майевтика** - рождение истины через вопросы
- **Ирония** - выявление противоречий в мышлении
- **Диалектика** - движение к истине через противоречия

## 🔍 Примеры многоуровневого диалога

### Уровень 1:
```
Утверждение: "Свободная воля существует"
Вопрос: "Как свободная воля совместима с детерминизмом?"
Ответ: "Свобода проявляется в выборе между предопределенными вариантами"
```

### Уровень 2:
```
Новое утверждение: "Свобода проявляется в выборе между предопределенными вариантами"
Вопрос: "Является ли выбор между предопределенными вариантами истинной свободой?"
Ответ: "Да, потому что сам факт выбора и есть свобода"
```

### Уровень 3:
```
Новое утверждение: "Сам факт выбора и есть свобода"
Вопрос: "А что если выбор тоже предопределен нейронной активностью?"
Ответ: "Тогда предопределенность выбора не отменяет его субъективную реальность"
```
