# 🔍 Система проверки противоречий

Игра теперь автоматически выявляет логические противоречия и корректно их оценивает в голосовании.

## 🧠 Как работает система

### **🔍 1. Автоматическая проверка после каждого ответа**
```
🛡️ Защитник: "Да, все животные имеют право на жизнь"

🔍 ФАЗА 4: Проверка противоречий
🚨 ОБНАРУЖЕНО ПРОТИВОРЕЧИЕ!
⚠️ СЕРЬЕЗНОЕ ПРОТИВОРЕЧИЕ (7/10)
Тип: Противоречие с изначальным утверждением
Объяснение: Ответ утверждает абсолютное право, но вопрос указывал на практические ограничения
```

### **🤖 2. Контекстная память для ИИ**
ИИ-игроки теперь получают полный контекст при голосовании:
- **Изначальное утверждение**
- **Полная история диалога** (все уровни)
- **Все обнаруженные противоречия** с оценками серьезности
- **Инструкции по типу личности** (скептик чувствителен к противоречиям)

### **🎯 3. Умное голосование**
```python
КРИТЕРИИ ОЦЕНКИ для ИИ:
1. Логическая последовательность (нет противоречий)
2. Качество ответов на вопросы
3. Соответствие изначальному утверждению
4. Честность и признание ограничений

ВАЖНО: Если есть серьезные противоречия (7+ баллов),
это должно сильно повлиять на оценку.
```

### **⚡ 4. Автоматические эффекты противоречий**

#### При обнаружении противоречий:
- **Серьезность 7+ баллов** → ИИ автоматически склоняется к "не убедил"
- **ИИ чаще хочет углубиться** → +10% вероятность за каждый балл серьезности
- **Показ противоречий** в финальном голосовании

#### Типы противоречий:
- **`self_contradiction`** - противоречие с собственными утверждениями
- **`statement_contradiction`** - противоречие с изначальным утверждением
- **`logical_fallacy`** - логическая ошибка в рассуждении

## 🔧 Техническая реализация

### **ContradictionChecker класс:**

```python
analysis = checker.check_for_contradictions(

    dialogue_history=[...]

# Результат:
{

    'explanation': 'Ответ слишком категоричен...',
    'severity': 8
}
```

### **Улучшенный промпт для ИИ:**
```
ИЗНАЧАЛЬНОЕ УТВЕРЖДЕНИЕ: Животные имеют права

ПОЛНАЯ ИСТОРИЯ ДИАЛОГА:
Уровень 1:
  Вопрос: В данный момент все животные имеют все права?
  Ответ защитника: Да, все животные имеют право на жизнь

ОБНАРУЖЕННЫЕ ПРОТИВОРЕЧИЯ:
- Ответ слишком категоричен для практической реальности (серьезность: 8/10)

ВОПРОС: Убедил ли защитник в итоге своими ответами?
```

## 🎮 Игровые эффекты

### **Для человека-игрока:**
- **Видите автоматический анализ** каждого ответа
- **Знаете силу своих аргументов** через оценку противоречий
- **Можете использовать системные противоречия** для углубления

### **Для ИИ-игроков:**
- **Скептик** особенно чувствителен к логическим ошибкам
- **Педант** требует строгой последовательности
- **Прагматик** оценивает практическую применимость
- **Синтезатор** пытается найти справедливый баланс

### **Стратегические преимущества:**
1. **Нахождение противоречий** → повышает шансы на победу
2. **Логическая честность** → ИИ это ценит
3. **Качественные вопросы** → автоматически выявляют слабости
4. **Системная поддержка** → не нужно полагаться только на субъективность

## 📊 Примеры противоречий

### **🚨 Критическое противоречие (9/10):**
```
Утверждение: "Свободная воля существует"
Ответ: "Всё предопределено нейронной активностью"
→ Прямое отрицание изначального утверждения
```

### **⚠️ Серьезное противоречие (7/10):**
```
Утверждение: "Животные имеют права"
Ответ: "Да, все животные имеют ВСЕ права"
→ Чрезмерное расширение изначального утверждения
```

### **⚡ Слабое противоречие (4/10):**
```
Утверждение: "Красота объективна"
Ответ: "Красота частично субъективна"
→ Незначительное смягчение позиции
```
