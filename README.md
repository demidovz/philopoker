# Философский Блеф

Многопользовательская игра для 5 игроков (1 человек + 4 ИИ-агента), где участники обсуждают философские утверждения, используя сократовский метод и механику ставок.

## Описание игры

В игре игроки обсуждают философские утверждения из различных областей философии:
- **Этика** ⚖️ - вопросы морали и нравственности
- **Метафизика** 🌌 - природа реальности и бытия
- **Эпистемология** 🧠 - теория познания
- **Логика** 🔬 - принципы рассуждения
- **Эстетика** 🎨 - философия искусства и красоты

## Правила игры

1. **5 игроков**: 1 человек + 4 ИИ-агента с разными типами личности
2. **5 раундов игры**
3. **Стартовый капитал**: 30 фишек у каждого игрока
4. **Роль философа** переходит по кругу каждый раунд

### Новая структура раунда

1. **Фаза утверждения**: Философ получает карточку и высказывает СВОЮ позицию (может не соглашаться!)
2. **Фаза позиционирования**: ВСЕ игроки (включая философа) ставят фишки на свою позицию
3. **Проверка единогласия**: Если все согласны или все против - все возвращают ставки
4. **Фаза дискуссии**: Меньшинство задает вопросы большинству
5. **Фаза переубеждения**: Большинство приводит аргументы
6. **Финальное голосование**: Все голосуют - убедительна ли аргументация большинства
7. **Распределение банка**: Если "да" - выигрывает большинство, если "нет" - меньшинство

## 🧠 Нейросетевые ИИ-личности (OpenRouter)

- **Скептик** 🤔 - критически анализирует утверждения, ищет противоречия
- **Педант** 📚 - требует точности определений и академической корректности
- **Прагматик** 🛠️ - оценивает практическую пользу и реальные последствия
- **Синтезатор** 🔄 - ищет компромиссы и объединяет разные точки зрения

> 💡 **Новое!** ИИ-игроки теперь используют OpenRouter для реалистичных философских дискуссий вместо простых алгоритмов.

## Установка и запуск

### Требования
- Python 3.8+
- OpenRouter API ключ (для нейросетевых ИИ)

### Быстрая установка
```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить API ключ
cp .env.example .env
# Отредактируйте .env и добавьте ваш OpenRouter API ключ

# 3. Запустить игру
python main.py
```

### Настройка нейросети
См. подробную инструкцию в [NEURAL_SETUP.md](NEURAL_SETUP.md)

### Тестирование
```bash
python -m unittest discover tests -v
```
